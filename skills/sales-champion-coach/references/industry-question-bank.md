# 多行业场景题库（起步版）

说明：
- 健身行业有原始知识库支持，题目最细。
- 其他行业为可直接使用的起步版场景题库，优先用于考核与模拟。
- 如果用户提供了更具体的产品、客单价、客群、渠道，再按用户上下文二次改写。

## 一、教育

常见关注点：效果、孩子情况、价格、时间安排、是否适合。

题目示例：
1. “我家孩子现在成绩也不算差，没必要现在就报吧？”
2. “你们课程太贵了，线上不是也能学吗？”
3. “我怕孩子坚持不下来，报了也是浪费。”
4. “我得回去和孩子爸爸商量一下。”

## 二、房产

常见关注点：预算、地段、时机、风险、家人意见。

题目示例：
1. “这个总价超我预算了，我再看看别的盘。”
2. “现在市场还不明朗，我不想这么快决定。”
3. “我老婆还没看，她不点头我定不了。”
4. “同样的面积，别家单价更低，你们凭什么贵？”

## 三、健身

常见关注点：时间、距离、价格、效果、教练专业度、能否坚持。

题目示例：
1. “我平时太忙了，根本抽不出时间来练。”
2. “你们离我有点远，我怕办了也坚持不下来。”
3. “我怕花了钱没效果，到最后白练。”
4. “私教课太贵了，而且我怕后面还有隐形消费。”
5. “我基础太差了，怕跟不上，也怕别人笑。”
6. “我先体验一下可以，但我不想被强推。”

## 四、软件 / SaaS

常见关注点：ROI、部署成本、上手难度、团队配合、替代方案。

题目示例：
1. “我们现在已经有系统了，没必要再折腾一套。”
2. “你这个功能听起来不错，但我担心团队不会用。”
3. “老板只关心投入产出，你怎么证明值这个价？”
4. “我先拿去给 IT 和老板看看，过两周再说。”

## 五、金融

常见关注点：风险、收益预期、合规、机构可信度、决策时机。

题目示例：
1. “我最怕的就是风险，你先别跟我讲收益。”
2. “你们和银行、券商到底有什么差别？”
3. “我不想现在做决定，先再观察一段时间。”
4. “我以前被坑过一次，现在对这类产品非常谨慎。”

## 六、医美

常见关注点：安全、效果、医生资质、恢复期、是否被坑。

题目示例：
1. “我就怕做了不自然，恢复期也耽误工作。”
2. “你们价格不低，我怎么知道不是营销包装？”
3. “我朋友说别家活动更划算，你们为什么贵？”
4. “我还想再看看医生和案例，不想现在定。”

提醒：在医美场景里，不要承诺具体疗效，不要输出违规宣传话术。

## 七、汽车

常见关注点：价格、贷款、配置、保值率、是否再等等。

题目示例：
1. “这台车配置是不错，但价格还是超我心理预期。”
2. “我怕现在买，过两个月就降价。”
3. “我再去别家店看看，顺便比比贷款方案。”
4. “我老婆更在意空间和安全，这台她未必会喜欢。”

## 八、旅游

常见关注点：价格、行程品质、时间安排、退改、安全与体验。

题目示例：
1. “你这个团费不便宜，我自己订是不是更省？”
2. “我最怕行程太赶，去了反而更累。”
3. “最近工作不确定，我怕报了又请不下假。”
4. “网上差评我也看了，我不太敢轻易定。”

## 九、餐饮（会员、储值、宴会、加盟或高客单服务）

常见关注点：性价比、口味、到店频率、信任、决策犹豫。

题目示例：
1. “储值听起来划算，但我怕最后用不完。”
2. “你们套餐不便宜，我怕家里人不一定都爱吃。”
3. “我先收藏一下，等有需要再来。”
4. “别家最近活动更猛，你们这个力度一般。”

## 十、装修

常见关注点：预算、施工质量、增项、工期、设计还原。

题目示例：
1. “装修最怕后期不停加钱，你们怎么保证不乱增项？”
2. “我预算有限，想做效果但又不想花太多。”
3. “我还想再看看几家设计方案再决定。”
4. “工期能不能稳？我最怕拖。”

## 十一、其他行业

如果用户行业不在以上范围，按下面模板现编场景：

### 通用模板 A：价格
“我觉得你们这个还是贵了，我得再比较一下。”

### 通用模板 B：时间
“我现在没空处理这件事，过段时间再说吧。”

### 通用模板 C：信任
“我还不太了解你们，先再看看。”

### 通用模板 D：需求模糊
“我不是很急，也还没想清楚要不要做。”

## 十二、按岗位微调题面

### 门店顾问
让题面更偏现场、面对面、临门一脚。

### 大客户经理
让题面更偏多角色决策、流程长、ROI 与风险。

### 电话销售
让题面更偏首句破冰、快速建立信任、争取下一步。

### 渠道拓展
让题面更偏合作收益、资源互换、合作门槛、政策透明。

## 十三、按训练重点调难度

### 异议处理
客户会更强势、更会比较、更会质疑。

### 快速成交
客户未必反感，但会拖、会犹豫、会说“再看看”。

### 高效共情
客户情绪更复杂，需要先理解再推进。

### 全面提升
把价格、时间、信任、需求模糊穿插组合，但仍保持“一题一个核心点”。
